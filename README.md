# TaskFlow — Облегченный веб-трекер задач 🚀

**TaskFlow** — это минималистичное и быстрое веб-приложение для управления повседневными задачами (MVP-версия). Проект разработан на языке Python с использованием микрофреймворка Flask и встроенной базы данных SQLite.

Идеально подходит для быстрой демонстрации навыков веб-разработки и работы с базами данных.

---

## 🛠️ Стек технологий

* **Backend:** Python 3, Flask (Маршрутизация, Обработка POST/GET запросов)
* **Database:** SQLite 3 (Реляционные таблицы, Автоинициализация при старте)
* **Frontend:** HTML5, CSS3 (Макет на Flexbox, Шаблонизатор Jinja2)

---

## 🔑 Ключевые возможности

1. **Автоматическая настройка (Zero Configuration):** База данных `database.db` и все таблицы генерируются бэкендом автоматически при самом первом запуске проекта. Никаких дополнительных логов и скриптов инициализации.
2. **Полный цикл CRUD:** Реализовано добавление (Create), чтение и вывод на экран (Read), а также удаление (Delete) задач из базы данных.
3. **Безопасность:** Использование параметризованных SQL-запросов со знаками вопроса (`?`) исключает возможность взлома базы через уязвимость **SQL-инъекций**.
4. **Современный UI:** Чистый интерфейс на базе Flexbox CSS с красивыми карточками задач и кнопками быстрого удаления.

---

## 📁 Структура проекта

```text
taskflow/
│
├── app.py               # Главный файл бэкенда (логика, маршруты, создание БД)
├── database.db          # Файл базы данных SQLite (генерируется сам при старте)
│
├── static/              # Папка со статическими файлами стилей
│   └── css/
│       └── style.css    # Оформление интерфейса (Flexbox CSS)
│
└── templates/           # HTML-шаблоны (Jinja2)
    └── dashboard.html   # Главная панель управления задачами
```

## 🚀 Инструкция по запуску

### 1. Запуск сервера
Откройте терминал в папке проекта и запустите приложение:
```bash
python app.py
```
*(Скрипт сам проверит наличие базы данных и нарежет таблицы, если файла `database.db` нет).*

### 2. Открытие в браузере
Перейдите по локальному адресу в любом браузере:
```text
http://127.0.0.1:5000
```
